# Info-card-
I have built a responsive info card just using HTML, CSS 

## Tech used.
1. HTML-5
2. CSS-3
3. CSS Flexbox
4. CSS Media Queries
   
## Guide.
In this repository, you will get two files in which all code is available. Download both files and run the HTML file to view the card. 
1. index.html
2. app.css
